URL Encoder & Decoder



What does this mean?

URL

URL means Uniform Resource Locator. A URL is simply the address of a website.

Encode URL

An encoded URL is simply text that is translated to be used in a URL.

Decode URL

A decoded URL is simply the meaning of the encoded URL.

Example

If the decoded text is "Carl Sagan's 'Meat Planet'" then the encoded text would be %22Carl%20Sagan's%20'Meat%20Planet'%22.
