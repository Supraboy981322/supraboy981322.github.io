Bookmarklet



What is a "bookmarklet?"

A bookmarklet is a browser bookmark that contains javascript instead of a URL. This allows you to execute javascript code in your browser just by clicking the shortcut in your bookmarks bar.




How do I add one?

You can add a bookmarklet in one of two ways.

Option 1

  • Copy the code
  • Create a bookmark (it doesn't matter what page you bookmark because we'll replace the URL with the code)
  • Edit the bookmark
  • Remove the URL for the bookmark and paste the code into the text box
  • Double check that code begins with "javascript: " (if it does not, add it to the beginning)
  • The URL text input box should now contain the code. (Ex. "javascript:document.body.contentEditable = 'true'; document.designMode='on'; void 0")
  • Save the changes

    • Option 3

    This method only applies if you have a link containing the code

  • If your bookmarks bar is visible, drag the link containing the code to the bookmark bar
  • If you cannot drag the link to your bookmark bar, your browser may have the option to barkmark it when right-clicking the link
