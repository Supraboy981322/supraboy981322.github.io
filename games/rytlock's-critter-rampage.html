Rytlock's Critter Rampage



About this game

Rytlock's Critter Rampage is a web game developed by Delly Sartika and the ArenaNet Cinematics team. The game started as a fake game as part of the Super Adventure Box commercial and the concept was developed into a full-fledged game. The game has many similarities to the Super Adventure Box, including hearts, enemies, mushrooms, and music.
