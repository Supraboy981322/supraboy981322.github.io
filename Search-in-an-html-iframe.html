Search In An HTML iFrame

Description

This is a website that allows the user to search in an HTML iframe. If you download the original file, you can browse using it without being seen by most anti-privacy software that teachers/employers use. This website was originally a single offline HTML document, but I decided to go public with it just for the fun of it. You can download the original file using the link further down the page

If you would like to download the original HTML file, please click here.

PLEASE NOTE:

- When using google, try using "https://www.google.com/webhp?igu=1" instead of "https://www.google.com/"

- Google images currently may not work

- If you find a YouTube or Google Images clone/mirror that works, please contact me with the link

- Your activity will still be attached to your IP Address, so please use a VPN if you insist on using this and don't want to be trackable

- Nothing you do here will show up on your browser history, but it will show up on government and business databases

- Feel free to use my code for the HTML iframe as you please, but make sure to give me credit for the code you decide to use.
